# Internal Communications Analytics

## Overview
This project analyses internal email communication patterns to understand factors influencing response rates and response times.

## Tools Used
- Power BI Desktop
- Excel

## Key Questions
- How responsive is internal communication?
- Does sender role affect response likelihood?
- Does subject length influence response speed?

## Key Insights
- Response rates vary significantly by sender role.
- Senior roles tend to receive higher response rates.
- Subject length shows limited correlation with response speed.
- Response times are widely distributed, indicating inconsistent norms.

## Deliverables
- Executive Overview dashboard
- Email Deep Dive analysis
- Exported PDF report for non-technical stakeholders

## Notes
All data used in this project is synthetic and created for portfolio purposes.
